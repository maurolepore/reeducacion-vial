# Preguntas del Examen Real de Reeducación Vial

Preguntas recordadas del examen presencial. Cubre aproximadamente la mitad de lo que se tomó.

## Multiple Choice

| Pregunta | Respuesta correcta |
|----------|--------------------|
| Bulbo para indicar parada de colectivos | (bulbo / señal luminosa de parada) |
| Qué indica una senda peatonal amarilla y blanca | Cruce peatonal para escolares |
| La bici senda está sobre la vereda y sirve para bici y monopatín eléctrico? | Sí |
| Las motos pueden circular por una ciclovía (margen izquierdo de una calle)? | No |
| Alguien en silla de ruedas eléctrica, puede considerarse peatón? | Sí |
| Qué son las macro manzanas? | Áreas para reducir la circulación vehicular y fomentar la circulación peatonal |

## Preguntas Libres (desarrollo)

| Pregunta | Temas clave |
|----------|-------------|
| Tres ejemplos de conducción solidaria | Ceder paso, respetar peatones, no obstruir intersecciones, etc. |
| Tres factores que influyen en el riesgo vial | Humano, vehicular, ambiental |
| Dos características de un conductor seguro y de un conductor peligroso | Atento/prudente vs. agresivo/distraído |
| Cómo afecta si el acompañante discute con el conductor o con alguien de afuera? Qué harías? | Genera distracción; pedir que pare la discusión, detenerse si es necesario |
| El uso del celular genera distracción. Qué consecuencias trae? Cómo lo usarías? | Aumenta riesgo de siniestro; usar manos libres, no manipular mientras se conduce |
