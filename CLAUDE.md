# Educación y Reeducación Vial - CABA

Información sobre el sistema de scoring y recuperación de puntos, material de estudio para exámenes de conducción y reeducación vial en Buenos Aires.

## Contenido

| Archivo | Descripción |
|---------|-------------|
| `README.md` | Cómo recuperar puntos (pago voluntario, reeducación, caducidad) |
| `practica.md` | Historial de exámenes de práctica |
| `sistemas-reid_manual-del-curso-de-reeducacion-vial.md` | Manual del curso (Sistemas Reid) |
| `manual_creandoconciencia_curso-de-reeducacion-vial.pdf` | Manual alternativo (Creando Conciencia) |
| `simulador-test-de-conducir/src/data/*` | Data que alimenta el simulador de examen del examen en CABA (https://bandinopla.github.io/simulador-test-de-conducir/) |
